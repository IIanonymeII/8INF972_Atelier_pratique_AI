Il est essentiel d'exécuter la commande 'pip install .\EdgeGPT\.' dans le répertoire 'api_gpt' !!!!!!!!

Car le programme dépend de la bibliothèque EdgeGPT présente dans le dossier EdgeGPT.
Cette bibliothèque a été modifiée en raison de problèmes de validité des signatures de conversation cryptées dans la version acutelle de Github. 
Une fois que vous avez effectué cette opération, les problèmes devraient être résolus.
